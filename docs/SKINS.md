# Skins

Supported skin styles:

- `elastic`
- `larry`
- `classic`
- `autumn_larry`
- `black_larry`
- `blue_larry`
- `green_larry`
- `grey_larry`
- `pink_larry`
- `plata_larry`
- `summer_larry`
- `teal_larry`
- `violet_larry`

The plugin detects the active Roundcube skin and loads the closest matching stylesheet. Unknown Larry-derived skins fall back to the Larry stylesheet. Other unknown skins fall back to Elastic.

Colored Larry palettes are lightweight CSS variable overrides and do not depend on external skin repositories.
